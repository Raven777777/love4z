# Love4z 💜

> **PLZ HUG ME HARD** — 一个充满情感的个人主页 / 交互式网页

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## ✨ 特性

- **Matrix 数字雨** — 动态黑客帝国风格背景
- **全屏翻页** — 滚轮 / 触摸 / 键盘均可操控
- **打字机效果** — 逐字展现内心独白
- **幽灵错误弹窗** — 复古 Win9x 风格错误消息
- **FPS 帧率监视器** — 实时性能检测
- **响应式提示** — 宽屏设备体验最佳
- **词云动画** — 兴趣爱好环绕展示

## 🛠 技术栈

- 原生 JavaScript (IIFE 模块化)
- 原生 CSS3 (径向渐变、动画、字体、响应式)
- HTML5
- WOFF2 自定义字体 (FOTMatisProB)

## 🚀 使用

直接在浏览器打开 `index.html` 即可，无需构建工具。

```bash
# 或通过本地服务器预览
npx serve .
```
